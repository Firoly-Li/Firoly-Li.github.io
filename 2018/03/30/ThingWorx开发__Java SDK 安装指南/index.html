ThingWorx开发__Java SDK 安装指南

Eclipse项目设置

创建Java项目

1、新建一个java项目

2、在New Java Project安装向导中，输入ThingWorx Development作为项目名称，然后单击Finish。

设置用户库

1、右键单击Package Explorer中新创建的项目。
2、展开Build Path，并选择Add libraries…

3、单击 User Library，然后点击 Next

4、单击User Libraries…

5、点击New，输入ThingWorx Java SDK作为库名称，然后单击OK

设置JavaDoc

1、选中ThingWorx Java SDK后，点击Add External JARs…

2、在弹出窗口中，选择下载的SDK中lib文件夹下的所有JAR，然后点击Open

3、在ThingWorx Java SDK库中，展开Thingworx-common-xxxx.jar文件夹并选择Javadoc位置。并点击右侧的Edit…

4、选择Javadoc in archive 和 External file 然后点击Browse…

5、在弹出的窗口中找到lib文件夹下的Thingworx-java-sdk JAR文件，然后选择Open

6、点击确定，选中ThingWorx Java SDK库中的复选框，点击Finish

导入示例文件

1、右键开发项目，然后单击import…

2、在导入向导中，将File System添加到过滤器，点击Next

3、在弹出窗口中，找到放置实例java文件的文件夹src。

4、点击OK，选中src文件夹的复选框，然后单击Browse…

5 、修改路径为ThingWorx Development/src

6、 连续点击ok-Finish，展开目录如下图表示导入示例成功
